# 校园二手书市 — 部署指南

## 方式一：本地打开（最快）

直接双击 `index.html` 在浏览器打开即可使用。数据存在浏览器的 localStorage 中。

## 方式二：GitHub Pages 部署（免费上线）

1. 在 github.com 注册账号并登录
2. 点右上角 + → New repository
3. 仓库名任意（如 `book-market`），选 Public
4. 上传 `index.html` 到这个仓库
5. 点仓库 Settings → Pages
6. Branch 选 `main`，文件夹选 `/ (root)` → Save
7. 等 1-2 分钟，访问 `https://你的用户名.github.io/book-market/`

## 方式三：Vercel 部署（更快）

1. 在 vercel.com 注册（用 GitHub 账号登录）
2. 点 Add New → Project
3. Import 你上传了 index.html 的 GitHub 仓库
4. Framework Preset 选 Other
5. Deploy → 几秒后就有公开链接

## 数据说明

当前数据存在浏览器本地（localStorage），清缓存会丢失。
后期如果要加后端（用真实数据库），找我升级 V2 版本。
