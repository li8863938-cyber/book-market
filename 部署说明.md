# 校园二手书市 — 部署指南

## 上线给同学们看

**需要一个 GitHub 账号**（没有的话去 github.com 注册，免费）

**步骤：**

1. 打开 https://github.com 注册并登录

2. 点右上角 `+` → `New repository`
   - Repository name: 填 `book-market`
   - 选 **Public**
   - 勾选 Add a README ❌（不要勾）
   - 点 Create repository

3. 创建后页面会显示几行命令，复制**第一段**（有 `git remote add origin` 的那行）
   - 类似：`git remote add origin https://github.com/你的用户名/book-market.git`

4. 回到你电脑上，打开终端（Win+R → cmd），依次执行：

```bash
cd C:\Users\Administrator\.openclaw\workspace\二手书市场
git remote add origin https://github.com/你的用户名/book-market.git
git branch -M main
git push -u origin main
```

5. 浏览器打开仓库页面 → Settings → Pages
   - Branch: 选 `main`
   - Folder: 选 `/ (root)`
   - 点 Save

6. 等 1-2 分钟，访问：
   `https://你的用户名.github.io/book-market/`

**搞定。** 把这个链接发到班级群，同学就能用了。
